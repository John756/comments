CommentController Resource

REST controller for managing comments. This controller provides endpoints for CRUD operations on comments. It uses CommentService to interact with the data layer. The API is exposed under the path `/api/comments`.

Note: This controller is annotated with Validated, which means that method parameters and return values are subject to validation.
